欢迎使用北邮Sakai远教版

北邮sakai远教版 基于最新的sakai11。sakai11带来如下新特性：

  • 自适应用户界面：全新的用户界面，完全自适应手机、PAD、电脑屏幕;
  • 新一代成绩册：提供更快、更易用的分数录入能力，支持电子表格导入;
  • 课程组织工具: 支持多列布局，优化内容编写和展示;
  • 公共广播系统：赋予sakai管理员向指定人群或 全系统范围 发送警示提示；
  • 智能反馈工具：提供4种反馈途径，只需一个点击，反馈工具自动发送反馈邮件，自动添加站点URL、浏览器和版本号、插件等信息；
  • 数学公式：无需安装插件，支持跨浏览器，自适应屏幕尺寸；
